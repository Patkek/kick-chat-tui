A lightweight terminal UI client for viewing Kick chat channels. Built in Rust with `ratatui`, this tool focuses on a clean, keyboard-friendly channel browsing experience.

## What it does

- View channel message history in a TUI interface
- Navigate channels and read chat content
- Minimal, fast, and designed for terminal use

## Limitations

- Beta software: expect changes and improvements
- You can view messages, but sending chat messages is not supported yet

## Built with

- Rust
- ratatui for terminal UI

## Getting started

1. Clone the repository
2. Build with:

```bash
cargo build --release
```

3. Run the app:

```bash
cargo run --release
```

## Notes

This is a TUI-first client and aims to keep the interface simple and responsive. Contributions and feedback are welcome.
